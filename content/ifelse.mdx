
Ehtolauseiden avulla ohjelma voi suorittaa vaihtoehtoisia asioita riippuen siitä onko jokin ehto tosi vai epätosi. Python-ohjelmointikielessä
ehtolauseita laaditaan if - else if - else -rakenteen avulla. Tässä osiossa tutustutaan ehtolauseiden käyttöön.

<Segment title="1. if-else">
Alla oleva ohjelma lukee käyttäjältä syötteen (merkkijono), jonka arvo sijoitetaan muuttujaan <code>name</code>. Syötteen lukemisen jälkeen tapahtuu
seuraavaa:

* Jos (<code>if</code>) muuttujan <code>name</code> arvo on yhtäsuuri kuin <code>Tuomas</code> tulostuu teksti "You're awesome!"
* ... muuten (<code>else</code>) tulostuu teksti "meh..." 

Testaa ohjelmaa eri syötteillä! Jos mielestäsi joku muu kuin Tuomas on awesome muokkaa koodia mieleiseksesi. Testaa myös onko isolla alkukirjaimella merkitystä

<Obs>
Huomasit kenties että if-lauseessa oli yhtäsuuruusmerkkinä <Code>==</Code>. Nyt täytyy ehdottomasti pysähtyä selventämään eräs tärkeä asia:

* <Code>=</Code> on <em>sijoitusoperaattori</em>, jota käytetään kun muuttujalle asetetaan arvo.
* <Code>==</Code> on <em>yhtäsuuruusmerkki</em>, jota käytetään <em>ehtolauseissa</em>, kuten edellisessä koodiesimerkissä.

Huomaa että jälkimmäinen tapa käyttää yhtäsuuruusmerkkejä vastaa matematiikan yhtäsuuruusmerkkiä.
</Obs>
<Obs>
Vertailuoperaattorit:

* yhtäsuuri kuin: <Code>==</Code> 
* pienempi kuin:<Code>&lt;</Code> 
* suurempi kuin: <Code>&gt;</Code> 
* eri suuri kuin: <Code>!=</Code> 
* pienempi tai yhtäsuuri kuin: <Code>&lt;=</Code> 
* suurempi tai yhtäsuuri kuin: <Code>&gt;=</Code> 

</Obs>

<Segment title="2. Useita vaihtoehtoja: If - else if - else">
Jos vaihtoehtoja on useita käytetään if - else if - else -rakennetta. Tutkaile ensin alla olevaa koodiesimerkkiä, miten kyseistä ehtolauserakennetta käytetään. Tässä vielä seikkaperäiset ohjeet:

* Ensimmäisenä tulee aina <code>if</code>
* Jos vaihtoehtoja on useita läiskytetään aina uusi <code>elif</code> jokaista uutta ehtoa varten (<code>elif</code> on lyhenne sanoista "else if").
* <code>if</code>:n ja <code>elif</code>:n tapauksessa tulee aina ehto esim. <Code>if nimi == "Tuomas":</Code>, <Code>elif x &lt; 10:</Code> tai  <Code>elif y != 6:</Code>
* Lopuksi laitetaan <code>else</code>, joka tarkoittaa "jokin muu kuin ylläolevat". Tähän ei tule ehtoa. 

Testaa alla olevaa ohjelmaa eri syötteillä. Toimiiko se niinkuin ajattelisit sen toimivan?
<Obs>
Huomaathan koodissa olevat puolipisteen, sekä sisennykset riveillä 7, 9, 11 ja 13. Sisennys tehdään tabulaattorinäppäimellä!
</Obs>
